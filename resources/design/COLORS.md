# Color Palette: LLM Resayil Portal

**Design System:** Dark Luxury - Gulf B2B Professional

## Primary Colors

| Color | Hex | Usage | Context |
|-------|-----|-------|---------|
| Deep Charcoal | `#0f1115` | Main background | Page backgrounds, main cards |
| Dark Navy | `#0a0d14` | Secondary background | Sidebars, nested cards |
| Gold Primary | `#d4af37` | Primary actions | Primary buttons, key highlights |
| Gold Light | `#eac558` | Hover states | Button hovers, subtle accents |
| Slate Border | `#1a1d24` | Borders | Card borders, input borders |

## Semantic Colors

| Color | Hex | Usage | State |
|-------|-----|-------|-------|
| Success | `#059669` | Success messages | Green |
| Warning | `#f59e0b` | Warning alerts | Amber |
| Error | `#ef4444` | Error states | Red |
| Info | `#3b82f6` | Informational | Blue |

## Text Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Text Primary | `#e0e5ec` | Main content text |
| Text Secondary | `#a0a8b5` | Labels, subtitles |
| Text Muted | `#6b7280` | Placeholder text, help text |
| Text Gold | `#d4af37` | Accent text, highlights |

## Gradient Colors

| Gradient | Usage |
|----------|-------|
| `#d4af37 → #eac558` | Primary buttons |
| `#0f1115 → #0a0d14` | Card backgrounds |
| `rgba(212, 175, 55, 0.1) → transparent` | Subtle overlays |

## CSS Variables

```css
:root {
  /* Primary */
  --color-primary: #d4af37;
  --color-primary-hover: #eac558;
  --color-bg-main: #0f1115;
  --color-bg-secondary: #0a0d14;
  --color-border: #1a1d24;

  /* Semantic */
  --color-success: #059669;
  --color-warning: #f59e0b;
  --color-error: #ef4444;
  --color-info: #3b82f6;

  /* Text */
  --color-text-primary: #e0e5ec;
  --color-text-secondary: #a0a8b5;
  --color-text-muted: #6b7280;
}
```
