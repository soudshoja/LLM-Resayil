# Typography: LLM Resayil Portal

**Design System:** Dark Luxury - Gulf B2B Professional

## Font Families

### Arabic (Primary Language)
- **Font:** Tajawal
- **Weights:** 400 (Regular), 500 (Medium), 700 (Bold), 900 (Black)
- **Purpose:** All Arabic text - body, headings, buttons

### English (Secondary Language)
- **Font:** Inter
- **Weights:** 400 (Regular), 500 (Medium), 600 (Semi-bold), 700 (Bold)
- **Purpose:** All English text - body, headings, buttons

## Google Fonts Import

```html
<!-- Arabic -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Tajawal:wght@400;500;700;900&display=swap" rel="stylesheet">

<!-- English -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```
