# Phase 09-05: Complete Website Translation

**Priority:** HIGH
**Estimated effort:** 6 waves, ~20 atomic tasks
**Depends on:** Phase 09-03 (Arabic RTL), Phase 09-04 (Language Switcher)
**Branch:** `dev`
**Audit reference:** `TRANSLATION-AUDIT.md`

---

## Objective

Bring the entire LLM Resayil website to full bilingual EN/AR parity. Every user-facing string must go through Laravel's `__()` translation system. Fix all structural bugs in existing lang files, clean up corrupted Arabic text, and add missing keys.
