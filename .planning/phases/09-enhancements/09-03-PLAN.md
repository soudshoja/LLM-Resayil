# Phase 09-03: Full Arabic RTL Implementation

## Objective
Implement complete Arabic RTL (Right-to-Left) support across the entire website. This includes all pages and components - EXCEPT LLM model names which should remain in their original English format (Llama, Qwen, Mistral, DeepSeek, Gemma, Phi, GPT, BGE, etc.).

## Current State
**What's already done:**
- RTL middleware working (`ar` locale detection)
- Locale route configured (`/ar/*` paths)
- `dir="rtl"` attribute on HTML root
- Tajawal font loaded for Arabic text
- Language files complete for: validation, navigation, auth, billing, profile

**What needs work:**
- All Blade view files need `__()` translation wrapping
- Missing Arabic translations for many text strings
- RTL-specific CSS adjustments for tables, forms, cards

## Implementation Plan

### Wave 1: Language Files & Core Translations

**Tasks:**
1. Create/update all Arabic language files in `resources/lang/ar/`
   - navigation.php - menu items, brand, labels
   - auth.php - login, register, OTP messages
   - dashboard.php - stats labels, model catalog text
   - profile.php - profile section titles, phone verification
   - billing.php - plans, payment methods, top-ups
   - legal.php - about, contact, privacy, terms pages
   - teams.php - team management translations

2. Add missing language keys to English files:
   - resources/lang/en/dashboard.php
   - resources/lang/en/profile.php
   - resources/lang/en/billing.php

### Wave 2: Blade View Translations

**Public Pages:**
1. `auth/login.blade.php` - sign in form, labels, errors
2. `auth/register.blade.php` - registration form, OTP flow
3. `dashboard.blade.php` - stats cards, model catalog, API keys table
4. `profile.blade.php` - profile info, phone verification, password change
5. `billing/plans.blade.php` - subscription tiers, top-up cards
6. `billing/payment-methods.blade.php` - payment methods list, add form
7. `credits.blade.php` - credits explanation page
8. `docs.blade.php` - documentation listing
9. `welcome.blade.php` - landing page content

**Admin Pages:**
10. `admin/dashboard.blade.php` - stats, users table, modals
11. `admin/api-settings.blade.php` - API key creation
12. `admin/models.blade.php` - model management
13. `admin/monitoring.blade.php` - monitoring page

**Teams Pages:**
14. `teams/dashboard.blade.php` - team management layout
15. `teams/components/*.blade.php` - add member, role badge, team table

**Error Pages:**
16. `errors/401.blade.php` - unauthorized error
17. `errors/403.blade.php` - forbidden error
18. `errors/404.blade.php` - not found error
19. `errors/500.blade.php` - server error

### Wave 3: RTL CSS Adjustments

**RTL-specific fixes needed:**
1. Update layout cards - flip margins/paddings for RTL
2. Fix tables - change text-align from left to right in RTL
3. Forms - adjust label/input positioning
4. Icons/arrows - reverse direction for RTL
5. Buttons - adjust icon positioning
6. Badges/borders - ensure proper alignment
