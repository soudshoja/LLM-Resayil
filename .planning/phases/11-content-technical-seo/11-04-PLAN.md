- Responsive: Stack on mobile, 2-col layout on large screens (optional)
- Accordion (collapsible) optional but recommended for UX

JSON-LD Schema structure (in <head>):
```php
@php
$faqs = [
    ['question' => 'How do I get started...', 'answer' => 'To get started... [full text]'],
    // ... 12-15 items
];

$schema = [
    '@context' => 'https://schema.org',
    '@type' => 'FAQPage',
    'mainEntity' => array_map(function($item) {
        return [
            '@type' => 'Question',
            'name' => $item['question'],
            'acceptedAnswer' => [
                '@type' => 'Answer',
                'text' => $item['answer']
            ]
        ];
    }, $faqs)
];
@endphp

<script type="application/ld+json">
@json($schema)
</script>
```

Avoid: Short one-liner answers (Google FAQPage needs substantial content), duplicate questions, broken links to /docs.
