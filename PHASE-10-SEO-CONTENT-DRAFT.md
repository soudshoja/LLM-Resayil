## 6. Additional SEO Recommendations

### Canonical Tags
Add to all pages (especially public ones) to prevent duplicate content issues:
```html
<link rel="canonical" href="{{ request()->url() }}">
```

### Hreflang Tags (Bilingual Support)
For pages available in both English and Arabic:
```html
<link rel="alternate" hreflang="en" href="https://llm.resayil.io/en/page">
<link rel="alternate" hreflang="ar" href="https://llm.resayil.io/ar/page">
<link rel="alternate" hreflang="x-default" href="https://llm.resayil.io/page">
```

### robots.txt Recommendations
```
User-agent: *
Allow: /
Disallow: /admin
Disallow: /dashboard
Disallow: /profile
Disallow: /api-keys
Sitemap: https://llm.resayil.io/sitemap.xml
```

## Notes for Implementation Team

1. **Bilingual Consistency:** All meta descriptions must have Arabic equivalents. Use i18n keys where possible.
2. **Image Sizes:** OG images should be 1200x630px (ideal for sharing); Twitter images 1024x512px.
3. **Canonical URLs:** Always include canonical tag to prevent duplicate content penalties.
4. **Schema Validation:** Test all schema markup with [Google's Rich Results Test](https://search.google.com/test/rich-results).
5. **Dev vs. Prod:** Use environment checks (`app()->isProduction()`) to serve different schema/analytics per environment.
6. **Future Blog:** When `/blog` launches, follow same meta/title format and include article schema.
