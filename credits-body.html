PRICING MODEL

One system. Zero confusion.

Other LLM providers charge per-model in USD with separate input and output rates. We simplified it.

Token-based pricing (other providers)

Complex & unpredictable

Different rates per model. Input tokens priced separately from output tokens. Prices in USD subject to exchange rates.

EXAMPLE: GPT-4o, 1 request
Input tokens: 30 tokens × $0.0000025 = $0.000075
Output tokens: 150 tokens × $0.000010 = $0.001500
Exchange (KWD): 0.307 KWD/USD
Conversion fees: + variable
~$0.001575 + fees

RATES

Credit rates by model tier

All models on the platform fall into one of two tiers. The model list at /api/v1/models shows which tier each model belongs to.

Model Tier Credits per Output Token Example Models Best For
Standard 1 credit Llama, Qwen, Gemma, Mistral General chat, coding, summarization
Premium 2 credits DeepSeek, large reasoning models Complex reasoning, long context, advanced tasks
Not sure which tier a model is? Call GET /api/v1/models — the response includes a credit_multiplier field showing 1 or 2.
